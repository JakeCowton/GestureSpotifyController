# Gestures

## Play

- Both arms up to the side

```
                 |
                 |
----------------------------------------
                 |
                 |
                 |
                 |
                 |
                 |
                 |

```

## Pause

- Both arms up in the air

## Skip forward

- Right arm up by side

```
                  |
                  |
            --------------------------
            |     |
            |     |
            |     |
            |     |
            |     |
            |     |
                  |

```

## Skip backward

- Left arm up by side

## Volume up

- Both arms pointing up, 90 degree bend at elbow

```
            |     |    |
            |     |    |
             -----------
                  |
                  |
                  |
                  |
                  |
                  |
                  |

```

## Volume Down

- Both arms pointing up, 90 degree bend at elbow

```
                  |
                  |
             -----------
             |    |    |
             |    |    |
                  |
                  |
                  |
                  |
                  |

```

## Mute

- One hand over mouth
